## Key Discoveries (written to wiki)

1. **Flux expression ceiling** — "nervous" always renders as "angry" (9+ rounds confirmed)
2. **Evaluator calibration** — rigid checklists miss holistic quality; human picked 3/10 image as final
3. **Multi-dimension scoring** — composite assets need per-region scores, not single number
4. **Sprite path gotcha** — `LoadAssetAtPath<Sprite>` fails in Assets/Art/, works in Assets/Resources/
5. **Knowledge routing** — "how to do?" → skill; "true for any game?" → base
6. **Wiki discipline** — every page write must update index + log + ≥2 wikilinks
7. **Dispatch parallelism** — single-threaded loop blocks Art behind QA; ThreadPoolExecutor fixes it
