## Key Discoveries

1. **claude -p can't precisely edit 1000-line C# files** — layout spec JSON
   is correct but LLM execution of anchor changes is imprecise. Need code
   generation script instead.

2. **Dispatch parallel execution works** — Art(ComfyUI) + Engineering(Unity)
   truly simultaneous after ThreadPoolExecutor fix.

3. **Self-review is non-negotiable** — human found 9 issues that checklist
   would have caught. Written as permanent rule.

4. **Industry research prevents bad design** — "kill the yellow divider"
   was obvious once we looked at 4 competitors.

5. **Tag by capability not genre** — "layout, touch-zone, inventory" works
   for card games AND tower defense AND RPG.

6. **Karpathy's 4 principles** — Think, Simplify, Surgical, Goal-Driven.
   Every one maps to a mistake we made today.
